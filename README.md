# mahjong_ai

麻雀AI

## 計画

1. (済)麻雀ができるようなプログラムを作成
2. 一人プレイを実装
   1. 暗カン
   2. ツモ上がり
   3. リーチ
   4. 点数を計算
   5. あがりの形の判定
3. 四人プレイを実装
   1. ポン/チー/明カン
   2. ロン
4. ルールベースを導入
5. DQNを導入
